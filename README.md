# Пульт охраны банка

этот проект является пультом охраны банка

### Как установить

после загрузки репозитория создайте файл `.env`

В файле должны быть следующие переменные:
- `SECRET_KEY` — ключ для работы Django
- `DB_PORT` — порт базы данных
- `DB_HOST` — адрес базы данных
- `DB_USER` — имя пользователя базы данных
- `DB_PASSWORD` — пароль пользователя базы данных
- `DB_NAME` — название базы данных

Python3 должен быть уже установлен.
На Python3.11.1 работоспособность приложения гарантирована.
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
после всех действий запустите терминал и введите "python main.py", пример запуска:
```
python main.py
```
