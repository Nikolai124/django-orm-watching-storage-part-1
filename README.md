# Пульт охраны банка

этот проект является пультом охраны банка

### Как установить

после загрузки репозитория создайте файл `.env`

В файле должны быть следующие переменные:
- `DJANGO_SECRET_KEY_DB` — ключ для работы Django
- `DJANGO_PORT_DB` — порт базы данных
- `DJANGO_HOST_DB` — адрес базы данных
- `DJANGO_USER_DB` — имя пользователя базы данных
- `DJANGO_PASSWORD_DB` — пароль пользователя базы данных
- `DJANGO_NAME_DB` — название базы данных

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
